Projects

Technical projects I've developed:

DBLP Statistics - Map Reduce Implementation

Used the Apache Hadoop Framework to perform parallel computation of various statistics on the raw Computer Science DBLP XML data. Additionally, migrated the implementation to AWS EMR. Built with Scala, Hadoop, Logback, SLF4J, Typesafe Configuration Library, Git, SBT, Cloudera QuickStart VM, and Mahout.

Chord - Cloud Overlay Network Algorithm with Consistent Hashing

Implemented and analyzed Chord algorithm using Akka HTTP based simulator, distributing resources over a resilient overlay network and accessing them using distributed hash tables. Built with Scala, Akka, Cloudsim, Logback, SLF4J, Typesafe Configuration Library, Git, SBT, and Docker.

Monte Carlo Simulations for Stock Portfolio in Apache Spark

Created simulations for testing trading strategies and recording losses and gains based on Monte Carlo methods. Used Apache Spark for distributing the data processing for the iterative algorithm, and then migrated the simulator to AWS EMR. Built with Scala, Apache Spark, Cloudera Quickstart VM, Logback, SLF4J, Typesafe Configuration Library, Git, SBT, and Docker.
